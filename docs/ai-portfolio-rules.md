# AI portfolio generation rules

These rules are hard-coded into the generation layer so every AI-created portfolio stays on-brand:

- Keep the portfolio mobile-first and readable within a few seconds.
- Preserve hero, about, highlight, contact, and footer identity structure.
- Surface contact actions clearly when the matching data exists.
- Never invent work history, education, metrics, awards, or client names.
- Never emit raw HTML, JavaScript, or unsafe content.
- Keep the tone premium, credible, and conversion-focused.
- Use only supported section types from the blueprint schema.
